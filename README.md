Requirements
------------

vasputils requires the following to run:

  * python 2.7
  * numpy

Installation
-----

```sh
pip install git+git://github.com/akiraakaishi/vasputils.git
```

Usage
-----

```sh
vasprun --eigenval -f vasprun.xml > eigen.data
```
